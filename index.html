Supabase 연결 설정

1. supabase.com에서 프로젝트를 만드신 뒤, Project Settings → API 메뉴에서
Project URL과 anon public key 두 가지를 복사해 아래에 붙여넣으세요.
2. SQL Editor에서 제공된 supabase_schema.sql 파일 내용을 실행해 표를 먼저 만들어주세요.
3. 아래에 입력 후 "연결하기"를 누르면 이 브라우저에 저장되어 다음에 다시 입력할 필요가 없습니다.
–
미완료 하자 (접수+확인+조치중)
–
긴급 우선순위
–
외주 처리중
–
이번달 완료
–
방막중 객실
–
총 방막일수 (누적)
–
반복하자 건수
–
