이 기능은 Supabase Edge Function을 통해 작동합니다.
아직 설정하지 않았다면 아래 Edge Function 배포 가이드를 참고하세요.

🛠️ Edge Function 배포 가이드

최초 1회만 진행하면 됩니다.

1단계  Supabase CLI 설치
npm install -g supabase
2단계  로그인 및 프로젝트 연결
supabase login
supabase link --project-ref 여기에-프로젝트-ID
3단계  Function 파일 배포 (함께 전달받은 폴더 기준)
supabase functions deploy invite-user --no-verify-jwt
4단계  환경변수 설정 (Supabase 대시보드)
app.supabase.com → 프로젝트 → Edge Functions → invite-user → Secrets 탭
SUPABASE_SERVICE_ROLE_KEY = (Project Settings > API > service_role key)
SITE_URL = https://seoungminlee.github.io/room-defect-dashboard
5단계 완료 후 "초대 발송" 버튼이 정상 작동합니다.
